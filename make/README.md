# Make.com blueprints

Importable Make.com scenario blueprints that use the ConnectSafely API via HTTP modules.

Get an API key: https://connectsafely.ai/api-key?utm_source=github&utm_medium=cookbook&utm_campaign=make

## Contributing

Each scenario lives in its own subfolder with:

- `blueprint.json` — exported via Make's "Export Blueprint" option.
- `README.md` — what it does and which connections / variables to set.
- `screenshot.png` (optional).

Strip API keys and connection IDs before exporting.
