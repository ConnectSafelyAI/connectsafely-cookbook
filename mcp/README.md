# MCP examples

Recipes for using the [ConnectSafely MCP server](https://connectsafely.ai/docs) from MCP-aware clients (Claude Desktop, Claude Code, Cursor, Continue, …).

This folder will host:

- Client config snippets (`.mcp.json`, `claude_desktop_config.json`, etc.)
- End-to-end prompts that exercise common tools (`fetch_profile`, `search_people`, `send_message`, …)
- Patterns for chaining MCP tools with other servers (web search, your CRM, etc.)

Get an API key first: https://connectsafely.ai/api-key?utm_source=github&utm_medium=cookbook&utm_campaign=mcp

## Contributing

See [../CONTRIBUTING.md](../CONTRIBUTING.md). MCP recipes should include the exact config block to drop into a client and at least one prompt that produces a verifiable result.
