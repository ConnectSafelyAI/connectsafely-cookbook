# Pricing & API access

API access is **paid only**. The "free" plan listed on https://connectsafely.ai/pricing is for the post-boosting product (one boosted post per week) and does **not** include API endpoints.

## API plan

**Ultimate Outreach — $10/month** ($0.33/day)

- Full LinkedIn API access for one connected LinkedIn account
- Documentation + community support
- Annual billing knocks roughly one month off

New API keys typically get a trial period; once it ends, all `/linkedin/*` endpoints will return `403 Forbidden` with `"Your trial has expired. Please subscribe to continue using this feature."` until you upgrade.

## Other plans

These exist for the auto-comment and post-boost products. They do not unlock additional API endpoints — API access is its own SKU.

| Plan | What you get |
| --- | --- |
| Free | Boost 1 post per week (no API) |
| Comment Starter — $29/mo | Auto-commenting, 1 LinkedIn account, up to 20 creators |
| Comment Pro — $59/mo | Auto-commenting at scale, up to 50 creators |
| Post Boost Starter — $19/mo | Daily AI-driven post boost |
| Post Boost Pro — $36/mo | 2 boosted posts/day with AI safety + replies |

For the canonical, current numbers see https://connectsafely.ai/pricing.
